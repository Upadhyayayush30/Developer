
## How to Run the Project

### Prerequisites

- Install Visual Studio Code (VS Code)
- Install Live Server extension in VS Code
- Ensure you have an internet connection to load external resources

### Steps

1. **Download the Project:**
   - Clone the repository or download the ZIP file and extract it.

2. **Open the Project in VS Code:**
   - Open VS Code.
   - Select `File > Open Folder...` and choose the `threejs-room` folder.

3. **Start Live Server:**
   - Right-click on `index.html` in the VS Code Explorer.
   - Select `Open with Live Server`.

4. **View the Project:**
   - Your default web browser will open and display the 3D room scene.
   - If you make changes to the `index.html` file and save them, Live Server will automatically refresh the browser to reflect the changes.

### Additional Notes

- The project uses external resources like textures and models from the Three.js examples. Ensure you have an internet connection to load these resources.
- You can add more GLTF models by modifying the `models` array in `index.html`.

Enjoy exploring and modifying your 3D room with Three.js!
